# welcome_message
a mcdrefoged plugin for PSS server

## Usage

`!!wm add` to add a welcome message

`!!wm delete` to delete a welcome message

`!!wm list` show welcome messages

`!!wm list <index>` show welcome messages use page index

## Dependencies

`mcdreforged` >=2.0.0
